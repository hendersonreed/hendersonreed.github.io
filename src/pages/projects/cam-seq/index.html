cam-seq, by reed

cam-seq won't really work on mobile, it requires a mouse and keyboard to operate effectively.

  • hit ? to toggle this help screen.

  • keys 0-9 select an instrument

  • keys qwertyuiop create note triggers at the mouse position based on the currently selected scale.

  • hover over a note trigger and hit d to remove it.

  • hit x to delete all note triggers

  • use the threshold slider to make all note triggers more or less sensitive
